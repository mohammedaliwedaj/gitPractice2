Introduction to terminal, Bash Script and Git

Main Bash Commands

  • pwd: Prints the current directory path.
  • cd: Changes the current directory.
  • ls: Lists directory contents.
  • mkdir: Creates a new directory.
  • touch: Creates a new file.
  • echo: Prints text, creates files, or writes text to files.
  • rm and rmdir: Removes files/directories.
  • grep: Searches for strings in files.
  • chmod: Changes file permissions.
  • exit: Closes the terminal window

Workflow: Cloning, Editing, Committing, and Pushing

  • Creating a Repository: Requires a GitHub account.
  • Cloning: Copies an existing repository to a local computer.
  • Committing: Saves changes to the local repository.
  • Pulling: Updates the local repository with changes from the remote repository.
  • Pushing: Uploads local changes to the remote repository
